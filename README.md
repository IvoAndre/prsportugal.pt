# PRS Portugal - Website Oficial 

Este repositório contém o código-fonte do website oficial da comunidade **PRS Portugal**. A plataforma serve como o ponto de encontro digital para atletas de tiro de precisão em Portugal, oferecendo informações sobre provas, regulamentos e a estrutura da modalidade.

## Funcionalidades
- **Calendário Dinâmico:** Listagem das próximas competições e eventos nacionais.
- **Guia para Iniciantes:** Secção dedicada a quem quer começar no PRS (equipamento, níveis, segurança).
- **Diretório de Patrocinadores:** Espaço dedicado aos parceiros que apoiam o desporto em Portugal.
- **Mobile Friendly:** Design responsivo adaptado para consulta rápida durante as provas.

## Tecnologias Utilizadas
- **HTML5 / CSS3:** Estrutura e estilização moderna.
- **JavaScript:** Funcionalidades interativas e manipulação de DOM.
- **GitHub Pages:** Alojamento e deployment contínuo.
